Generate YAML config file for SpPrAn pipeline

This form creates a config file for the SpPrAn pipeline.

Adjust the configurations to your needs filling the fileds. Then, download the config file.
After created, you can edit it directly in a code editor or using the Jupyter Notebook "edit_config_file.ipynb".

Add the pathnames to the directories and select the files' format type (tsv or csv).

Added protein markers should be named exactly as in the "Positivity" columns in the csv files.

Name the cell types you need and then define them selecting a state for each protein marker: "Present", "Abscent", "Not applicable"

Each time the pipeline is run, new output files are generated, but they are not automatically replaced with existing files unless you decide it.

Anndata files are used for a faster pipeline rerun since they contains cleaned neccesary data for generate outputs.
